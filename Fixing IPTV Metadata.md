 "tv_archive_duration": "7"
}

Full attribute list:

| Attribute             | Meaning                       |
| --------------------- | ----------------------------- |
| `num`                 | index number                  |
| `name`                | channel name                  |
| `stream_type`         | live / vod / series           |
| `stream_id`           | unique stream id              |
| `stream_icon`         | channel logo                  |
| `epg_channel_id`      | EPG id (equivalent to tvg-id) |
| `added`               | timestamp when added          |
| `category_id`         | category reference            |
| `custom_sid`          | custom stream id              |
| `tv_archive`          | catch-up enabled              |

| `tv_archive_duration` | catch-up days                 |
